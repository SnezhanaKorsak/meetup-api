# Meetup API
### CRUD REST Web API

Функционал Web API
1. Получение списка всех митапов;
2. Получение определённого митапа по его Id;
3. Регистрация нового митапа;
4. Изменение информации о существующем митапе;
5. Удаление митапа.

Информация о митапе
1. Название / тема;
2. Описание;
3. Набор тегов / ключевых слов;
4. Время и место проведения.
  
Стек
1. Node.js;
2. Express;
3. PostgreSQL 14.
